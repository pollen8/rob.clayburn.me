The Project

This highly abitious project aims to provide a real time collaborative environment for managing, sharing and annotating various media types.

My work

For the realtime part of this applicaiton I designed a node.js and Socket.io server side application.

Collaborative drawing The collaborative drawing app allows each user to draw, and see other users' drawings in realtime. Drawing is managed by a multi-layered HTML canvas object.

Collective writing was achieve using operational transformations, a technology similar that used by Google docs.

Syncronised video playback - the node.js server acts as a conductor ensuring that when a video is played all users see the same frame at the same time. As with the collaborative drawing, users could then draw directly over the video.
