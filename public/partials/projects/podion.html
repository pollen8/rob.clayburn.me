I am the lead developer for this stylish events management web app.

The initial version of this application was created using Fabrik, you can see an example of it running at ecicultuurfabriek.nl. The next iteration which will be out in the fall of 2014 is a complete rebuild, the whole application has been redesigned into three separate sections, to create a flexible content solution.

  • Administration - an angular.js web app, with a mini app for managing social media via buffer.
  • Content server - a REST server which stores events and serves the various views.
  • Front end application - a light weight front end application populated with data served from the content server.
