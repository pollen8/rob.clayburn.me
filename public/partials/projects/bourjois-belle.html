Bourjois Belle

The Project

Slingshot were contracted by MEC to produce a street game to promote the French makeup company Bourjois, during London Fashion week.

The delivered GPS based game allowed Fashion Week attendees to "hunt" for the Bourjois model, once found they could ask her a special question to unlock a special prize.

In parallel with the street based game, we also delivered a Facebook application where player's not based in London could guess the belle's initial position to win prizes.

My work
  • PHP
  • Facebook
  • Google maps
  • GPS
  • HTML
  • CSS

I was tasked with developing the Facebook application which took GPS generated data and mapped the Belle's locations onto a Google Map. The GPS data stream was rendered using polylines onto the map so that the Belle's route could be seen.

Player scores were recorded in a mySQL database and displayed as part of the Facebook application, these scores were then also used to select the winning participants
