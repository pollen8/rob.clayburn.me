The project

Joomla! is one of the world's top 3 open source content management systems. Part of the project's immense success is the amazing choice of available extensions. The Joomla! Extension Directory (JED) is the defacto location to find and download over 8000 extensions.

The Joomla! team were looking for a redesign of the site, the brief being twofold, first to integrate their current set of disperate administration solutions into a concise and user friendly interface for the JED team. Secondly, the front needed to be redesigned and simplified with the implementation of a new commenting and rating system.

My work

The Fabrik team were chosen to redesign the new Joomla Extension Directory.

Both the user facing and administration sections run off the same engine. Whilst the front end is rendered as individual web pages, the administration was built as a single page Angular.js application.

Expect the new JED site to be released in the fall of 2014.
