Asylum - 2.8 hours later

The project

2.8 Hours later is probably the largest street game running in the UK, and its with zombies - what more can you ask for! They needed someone to implement a responsive web site design

My work
  • HTML
  • LESS
  • Boostrap
  • Facebook API
  • Twitter API

Slingshot tasked me with implementing a complex site design to look as equally compelling on a PC monitor as on your phone or tablet, this was achieved with the bootstrap framework and a judicious use of media queries

There is also the possibility to log in to the site via Facebook or Twitter, and volunteer to help out during the games, a rare chance to become a zombie for the weekend!
